## Introduction

This guide explains how to use ComfyUI to create a pipeline and automatically send the output to Xiaohongshu.

## Prerequisites

Before you begin, ensure you have the following:

- ComfyUI installed and configured
- Xiaohongshu account

## Steps

### Step 1: Create a Pipeline in ComfyUI

1. Open ComfyUI.
2. Create a new pipeline by clicking on the "New Pipeline" button.
3. Add the necessary nodes to your pipeline to process the data as required.
4. Configure the nodes with the appropriate settings.

### Step 2: Configure Xiaohongshu Integration

1. In ComfyUI, go to the "Integrations" section.
2. Select "Xiaohongshu" from the list of available integrations.
3. Enter your Xiaohongshu account credentials and authorize the integration.

### Step 3: Set Up Automation

1. In the pipeline settings, enable the "Automatic Send to Xiaohongshu" option.
2. Configure the automation settings, such as the frequency and conditions for sending the output to Xiaohongshu.

### Step 4: Test the Pipeline

1. Run the pipeline to test the integration.
2. Verify that the output is automatically sent to Xiaohongshu as configured.

## Conclusion

By following these steps, you can use ComfyUI to create a pipeline and automatically send the output to Xiaohongshu. This integration allows you to streamline your workflow and ensure that your content is consistently shared on Xiaohongshu.
